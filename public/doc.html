Disc Golf App: ReadMe

GET /players

Returns all players in the database by id, firstName, lastName and gender

GET /players/:id

Gets a single player by his/ her id inside the path parameter

POST /players

Creates a player. .

DELETE /players/:id

Deletes a player by his / her Id

PUT /players/:id

Updates player inside the player's table by either their firstName, lastName, or gender all at once or individually that is up to the client.
