Disc Golf App ReadMe

GET /players

Returns all players in the database

GET /players/:id

Gets a single player by his/ her Id

POST /players

Creates a player and inserts it into the database inside the players table

DELETE /players/:id

Deletes a player by his / her Id

PUT /players

Updates player inside the players table
