Disc Golf App ReadMe

GET /players

Returns all players in the database by id, firstName, lastName and gender

GET /players/:id

Gets a single player by his/ her Id, and displays their object of id, firstName, lastName, and gender

POST /players

Creates a player and inserts information from the request body. Once new player is created we send a response that includes the new player object.

DELETE /players/:id

Deletes a player by his / her Id

PUT /players

Updates player inside the players table by their id
